Ambxst now ships with [axctl](https://github.com/Axenide/axctl), and a large part of the Hyprland service layer has been ported to it so we can start building support for other compositors.

#### Add

- New axctl daemon/service with state streaming, focus history, and window geometry.
- Compositor-agnostic FocusGrab and service layer built on AxctlService.
- Assistant sidebar with AI panel, model selector, code blocks, and image attachments.
- Compositor keybinds/config pipeline and keystore-backed AI credentials.

#### Change

- Compositor configs/presets moved from Hyprland-specific to generic compositor files.
- Shell, bar, dock, overview, and tools migrated to AxctlService.
- Dashboard compositor settings updated for the new service layer.
- AI provider strategies refreshed and expanded (Anthropic, Groq, Ollama, Mistral, OpenAI, Gemini).
- Installer and Nix packages updated to include axctl dependencies.

#### Fix

- Screenshot reliability and bar reservation/pinned state handling.
- Compositor sync issues and brightness edge cases.
- Reservation/sidebar glitches across shell and frame components.
- Optional chaining compatibility issues across several QML components.

#### Performance

- Lazy-load heavy views and multimedia components to reduce startup cost.
- Defer OSD and service initialization to improve boot time.
- Reduce mpv IPC chatter on static wallpapers.

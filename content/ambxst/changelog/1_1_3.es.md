#### Características

- Sistema de atajos de teclado reescrito desde cero para ser más flexible y fácil de configurar.
- Nueva utilidad de instalación: `ambxst install hyprland` (pronto más compositores).

#### Mejoras

- Panel de atajos de teclado mejorado con mejor visualización y experiencia de usuario.
- La configuración del compositor (de momento solo Hyprland) ahora es estática para mayor persistencia, estabilidad y flexibilidad.

#### Correcciones

- Arreglado un bug donde los pines de aplicaciones en la barra no se guardaban.
- Arreglado el script de instalación.

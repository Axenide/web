#### Features

- Rewritten keyboard shortcuts system from scratch to be more flexible and easier to configure.
- New installation utility: `ambxst install hyprland` (more compositors soon).

#### Improvements

- Improved keyboard shortcuts panel with better visualization and user experience.
- Compositor configuration (currently only Hyprland) is now static for better persistence, stability and flexibility.

#### Fixes

- Fixed a bug where application pins on the bar were not being saved.
- Fixed the axctl installation script.

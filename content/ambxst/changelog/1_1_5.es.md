#### Agregado

- Notificaciones de batería para niveles bajos y críticos, incluyendo una acción para activar ahorro de energía.
- Soporte para instalar Ambxst en configuraciones `hyprland.lua`, manteniendo compatibilidad con `hyprland.conf`.

#### Mejoras

- El lanzamiento de aplicaciones y archivos de escritorio ahora limpia el token de workspace inicial de Hyprland, para que las ventanas nuevas abran en el workspace activo.
- Los atajos predeterminados y la configuración generada del compositor ahora usan dispatchers normalizados para enfocar y mover ventanas entre layouts.
- El launcher de aplicaciones y la ejecución de archivos de escritorio ahora comparten un flujo de lanzamiento desacoplado más limpio.

#### Arreglos

- Los widgets de calendario ahora se actualizan correctamente cuando cambia la fecha.
- El espaciado del frame ahora se comporta correctamente con contenido en pantalla completa.
- La limpieza de notificaciones es más robusta cuando las notificaciones se reemplazan o se cierran.

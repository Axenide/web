#### Added

- Battery notifications for low and critical levels, including a power saver action.
- Support for installing Ambxst into `hyprland.lua` configs, while keeping `hyprland.conf` support.

#### Improvements

- App and desktop launches now clear the Hyprland initial workspace token, so new windows open in the active workspace.
- Default keybinds and generated compositor config now use normalized focus and window-move dispatchers across layouts.
- The app launcher and desktop file execution paths now share a cleaner detached launch flow.

#### Fixes

- Calendar widgets now refresh correctly when their date changes.
- Frame spacing now behaves correctly while fullscreen content is active.
- Notification cleanup is more robust when notifications are replaced or closed.
